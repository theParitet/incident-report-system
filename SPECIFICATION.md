# Specification

> [!IMPORTANT]
> This specification is not set in stone.

## Feature Description

The report system works around 3 core elements:

- Table of data
- Incident report creation form
- KPIs
  - KPIs Cards
  - Map Graph
  - Pie Graph
  - Timeline Graph

### Table Content

- Title: arbitrary string
- Location: arbitrary string
- Type: enum
- Priority: enum
- Status: enum
- Reporter: arbitrary string
- Data/time: arbitrary date
- Description: arbitrary string (textarea)
